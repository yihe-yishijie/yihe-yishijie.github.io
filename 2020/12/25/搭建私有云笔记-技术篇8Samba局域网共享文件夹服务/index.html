搭建私有云笔记-技术篇8Samba局域网共享文件夹服务

2020-12-25

关键词：IT，树莓派4b，samba，局域网共享文件夹

安装环境：
硬件，树莓派4b
操作系统，Raspbian GNU/Linux 10 (buster)

安装 Samba

sudo apt-get install samba samba-common-bin
配置 Samba
sudo nano /etc/samba/smb.conf
末尾加入

设置文件权限

添加 Samba用户并设置密码
sudo smbpasswd -a 用户名

重启
sudo samba restart

windows10开启

修改注册表：
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanWorkstation\Parameters]项，在右侧空白处右击鼠标，从其右键新建-“DWORD值（32位）”，重命名为“AllowInsecureGuestAuth”，同时将其值修改为“1”
“网络和共享中心”点击左上角的“更改高级共享设置”“来宾或公用”栏目，勾选“启用网络发现”和“启用文件和打印共享”项
运行“services.msc”“Server”服务设置为“自动”，并“启动”
重启
