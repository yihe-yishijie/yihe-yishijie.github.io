搭建私有云笔记-技术篇6linux硬盘挂载

2021-01-02

查看硬盘

sudo fdisk -l

查看挂载

sudo df -h

格式化

我们使用 parted 工具对对硬盘进行分区操作后，我们一定要进行格式化分区才能被使用！
使用 mkfs 工具可以方便地对分区进行格式化。

sudo mkfs -t ext4 /dev/sda

查询UUID命令

sudo blkid /dev/sda
其中/dev/sda为用sudo fdisk -l命令查询的硬盘（u盘）设备名称

临时挂载，每次关机后挂载会消失，例：
sudo mount -t ext4 /dev/sda /mnt/yishijie

开机挂载，注意uuid会变

sudo nano /etc/fstab

注意：
1.硬盘uuid的变化，
2.挂载文件夹类型用none，在hexo安装后挂载
